To add this widget to your existing dashboard:

  1. Include the TeoCoin widget JavaScript file
  2. Add a container div with the ID "teocoin-withdrawal-widget"
  3. The widget will automatically initialize

HTML Integration:

<!-- Include the TeoCoin widget script --> <script src="/static/js/teocoin-widget.js"></script> <!-- Add this div where you want the widget --> <div id="teocoin-withdrawal-widget"></div>

Dashboard API Integration:

The widget automatically works with your existing dashboard APIs:

  • /api/v1/dashboard/student/ - Student dashboard
  • /api/v1/dashboard/teacher/ - Teacher dashboard
  • /api/v1/dashboard/admin/ - Admin dashboard
